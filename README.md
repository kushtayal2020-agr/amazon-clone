# Amazon Homepage Clone

This project is a clone of the Amazon homepage built using **HTML and CSS**.
It replicates the basic layout and design of the Amazon website to practice frontend development and layout structuring.

## 🚀 Features

* Amazon-style navigation bar
* Search bar UI
* Hero banner section
* Product category cards
* Flexbox-based layout
* Hover effects on navigation items

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox

## 🎯 Purpose of the Project

The main goal of this project is to practice **frontend layout design**, improve understanding of **CSS positioning and Flexbox**, and recreate a real-world website interface.
